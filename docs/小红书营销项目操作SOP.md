# 小红书营销项目一体化平台操作 SOP

## 1. 启动工作台

1. Windows 双击 `start_ops.bat`；Mac 执行 `./start_ops.sh`。
2. 浏览器打开 `http://127.0.0.1:8081/ops-config`。
3. 进入「项目大盘」，先点击「检测当前电脑环境」。
4. 如果出现「需处理」，按右侧处理办法处理后重新检测。

## 2. 新建项目

1. 在「项目大盘」填写项目名。
2. 粘贴客户可访问的 Base 链接或 Base Token。
3. 点击「一键新建项目」。
4. 等待系统回填 Base、爆款表、评论表、达人智能圈选表、合作监控表等表 ID。
5. 在「项目检索栏」保存当前项目配置，后续可直接切换。

## 3. 平台爆款检索

1. 进入「平台爆款检索」。
2. 填写搜索关键词、最大笔记数、排序、笔记类型和发布时间。
3. 点击「启动爆款抓取」。
4. 抓取结束后系统自动同步爆款笔记、笔记二创表和爆款评论表。
5. 可在右侧结果浏览区查看标题、正文、图片和互动数据。

## 4. 达人智能圈选

1. 进入「达人智能圈选」。
2. 首次使用先点击「打开蒲公英登录窗口」，在浏览器中完成登录。
3. 填写达人昵称；小红书号只在重名时填写。
4. 点击「分析达人并列出相似达人」。
5. 如需补充相似达人详情，勾选相似达人后点击「抓取勾选相似达人详情」。
6. 结果会自动同步到达人智能圈选表。

## 5. 账号内容监测

1. 进入「账号内容监测」。
2. 粘贴样本账号主页链接或账号 ID，每行一个。
3. 设置每个账号抓取篇数。
4. 点击「启动样本账号抓取」。
5. 抓取完成后系统自动同步到账号筛选表。

## 6. 笔记数据监控

1. 进入「笔记数据监控」。
2. 填写合作博主账号名单。
3. 设置监控周期、每个账号抓取篇数和同步上限。
4. 可点击「抓取并同步一次」进行手动检查。
5. 稳定后点击「启动定时监控」。

## 7. 笔记舆情监控

1. 进入「笔记舆情监控」。
2. 设置登录方式、浏览器显示模式、评论抓取和二级评论抓取。
3. 二级评论指楼中楼评论；数据量较大时建议先关闭。
4. Cookie 登录仅在扫码登录不可用时使用，且只填写客户自己的 Cookie。

## 8. 常见问题处理

| 问题 | 处理办法 |
| --- | --- |
| 蒲公英窗口打不开 | 先运行交付环境检测；Mac 需安装 Chrome 或 Edge，或在 `config/base_config.py` 设置 `CUSTOM_BROWSER_PATH` |
| 飞书同步失败 | 确认 `lark-cli` 已安装，并执行 `lark-cli auth login` 使用客户飞书账号授权 |
| Playwright 浏览器缺失 | 重新运行启动脚本；仍失败时执行 `python -m playwright install chromium` |
| 页面提示未配置表 ID | 回到「项目大盘」重新点击「一键新建项目」，确认项目映射里每张表都有链接 |
| 抓取一直等待 | 查看「交付环境与运行日志」，复制最后 20 行日志给 AI 排查 |
