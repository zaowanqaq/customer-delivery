小红书运营配置（V1）

按流程操作：1初始化项目 -> 2样本账号抓取 -> 3爆款抓取 -> 4同步多维表 -> 5合作监控

项目切换器

用于已有项目 1/2/3 快速切换配置，不需要每次重新建表。

步骤1：初始化项目

只填项目名，系统自动新建 Base 并创建项目主表、账号筛选表、爆款监控表、合作监控表。

公共抓取配置（步骤2/3/5共用）

所有抓取任务共用同一套登录和评论参数。

步骤2：样本账号抓取（3-5账号）

输入样本账号主页链接或账号ID，每个账号默认抓 20 篇。

步骤3：爆款抓取（关键词）

用于项目关键词爆款检索，抓取后在步骤4写入多维表。

抓取进度监控

支持手动刷新与自动轮询（默认2秒），用于查看运行状态与最新日志。
