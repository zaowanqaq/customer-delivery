# MediaCrawler 本机版用户说明书

## 1. 这套系统现在能做什么
- 在本机打开运营页面并配置抓取参数。
- 支持按关键词抓取小红书内容。
- 支持筛选参数：排序、笔记类型、发布时间。
- 支持从飞书多维表“筛选规则”自动读取规则并启动任务。
- 抓取结果可保存为 CSV/JSON/JSONL/Excel 等格式。

## 2. 启动方式（推荐）
1. 双击运行 `start_ops.bat`。
2. 浏览器打开 `http://127.0.0.1:8080/ops-config`。
3. 在页面填写参数后点“启动任务”或“按规则表启动”。
4. 抓取结束后点“同步笔记到多维表”，把本地最新数据写入飞书表。

## 3. 页面字段说明
- 关键词：多个关键词用英文逗号分隔。
- 最大笔记数：控制本次任务最多抓取多少条笔记（可在页面直接改）。
- 登录方式：
- `qrcode`：会弹出浏览器扫码登录。
- `phone`：走手机号验证码登录。
- `cookie`：使用页面里填写的 Cookies 登录（不弹扫码）。
- 排序：综合、最新、最多点赞、最多评论、最多收藏。
- 笔记类型：不限、视频、图文。
- 发布时间：不限、一天内、一周内、半年内。
- 搜索范围、位置距离：当前为预留字段，先保存和透传，抓取侧暂未生效。
- Cookies：仅在 cookie 登录方式下需要填写。
- 同步 Base Token / 同步 notes 表 ID：用于“抓取后写入多维表”。
- 同步数量上限：控制每次同步到多维表的最大行数，`0` 表示全部。

## 4. 按规则表启动
需要先在飞书多维表里准备“筛选规则”表，并至少有一条启用规则：
- 启用：是/true/1
- 关键词：必须有值
- 排序、笔记类型、发布时间：按需填写

页面中填写：
- 规则表 Base Token
- 规则表 Table ID
- 指定规则名称（可空，空时自动取首条启用规则）

然后点击“按规则表启动”。

注意：
- “规则表”只用于控制抓取条件，不是结果明细表。
- 你提供的 Base（`N9BnbDq0WaDO4Fsrk8JcKNyUnie`）里，`trial_notes` / `trial_comments` 是结果表；若要按规则启动，请单独提供“筛选规则”表的 `Base Token + Table ID`。

## 5. 数据输出位置
- 默认在项目目录的 `data` 下按平台落盘。
- 小红书 CSV 常见路径：`data/xhs/csv/`
- 同步默认读取最新 `jsonl` 文件（更稳，不容易受 CSV 脏行影响）。
- 为对齐你的结果表字段，抓取落盘会同时输出兼容字段别名，例如：
- `title` + `标题`
- `user_id/nickname` + `author_user_id/author_nickname`
- `author_homepage_url`（由 `author_user_id` 自动拼接）
- 评论侧：`user_id/nickname` + `comment_user_id/comment_user_nickname`
- 本机抓取后可直接批量写入飞书 `trial_notes/trial_comments`（无需远程服务器）。

## 6. 常见问题
- 页面打不开：确认 `start_ops.bat` 运行窗口还在。
- 启动失败提示已在运行：先点“停止任务”再重试。
- 规则表读取失败：检查 `lark-cli` 是否安装并已授权。
- 抓取卡在登录：按页面登录方式完成扫码或补充可用 cookie。
- 运行时间长：通常是因为“最大笔记数较大 + 抓评论 + 每步默认等待 2 秒 + 登录验证耗时”。
- 想加快速度：把“最大笔记数”调小、关闭“抓取评论”、优先用 `jsonl` 保存并再同步到多维表。

## 7. 登录态说明（是否会记住）
- 会记住。默认 `SAVE_LOGIN_STATE=True`，程序会保存浏览器登录态，后续运行可复用。
- 记住方式：
- `qrcode/phone`：登录成功后，登录态保存在本地浏览器用户目录（`USER_DATA_DIR`）。
- `cookie`：使用你在页面填的 Cookies 直接登录；是否持久主要取决于 Cookie 本身有效期。
- 在当前默认 CDP 模式下，会使用真实浏览器 + 本地用户数据目录，通常更容易复用已登录状态。
- 失效场景：平台风控、Cookie 过期、清理本地用户数据目录、切换机器。
- 新电脑迁移后通常需要重新登录一次，这是正常现象。

## 8. 安全建议
- 不要在代码里保存服务器密码、Token、Cookie。
- 规则表 Token 建议只保存在本机配置，不要对外公开。
- 抓取账号建议专用，降低风控影响。
